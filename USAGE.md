# 사용 가이드

## 데모 사이트 기능

이 데모는 **WISE 법률 리서치 어시스턴트**의 주요 기능을 보여줍니다.

### 🎬 중요: 데모 모드

- **모든 응답은 미리 설정된 샘플 데이터입니다**
- 어떤 질문을 입력해도 동일한 응답이 반환됩니다
- 실제 AI 분석이나 데이터베이스 검색은 수행되지 않습니다
- 파일 업로드 기능은 UI만 동작합니다 (실제 분석 X)

## 주요 기능

### 1. 판례 검색 (Precedent Search)

**사용법**:
1. 서비스 선택: "판례 검색"
2. 질문 입력 (예: "부당해고 관련 판례를 찾아주세요")
3. 전송 버튼 클릭

**데모 응답**:
- AI 기반 법률 분석 결과
- 관련 판례 3건 (대법원, 고등법원 판결 등)
- 법률적 검토 및 권고사항
- 재정의된 질문 3개

### 2. 수임료 예측 (Fee Prediction)

**사용법**:
1. 서비스 선택: "수임료 예측"
2. 사건 내용 입력
3. 전송 버튼 클릭

**데모 응답**:
- 착수금 예상 금액
- 성공보수 예상 금액
- 총 예상 비용 범위
- 비용 산정 근거

### 3. 소장 생성 (Complaint Generation)

**사용법**:
1. 서비스 선택: "소장 생성"
2. 사건 개요 입력
3. 전송 버튼 클릭

**데모 응답**:
- 완성된 소장 문서
- 청구취지, 청구원인, 입증방법 포함
- 실제 법률 문서 형식

### 4. 의뢰서 초안 (Draft Request)

**사용법**:
1. 서비스 선택: "의뢰서 초안"
2. 상황 설명 입력
3. 전송 버튼 클릭

**데모 응답**:
- 법률 자문 의뢰서 초안
- 의뢰인 정보, 사건 개요, 자문 사항 포함

## UI 기능

### 대화 기록 (좌측 사이드바)

- **새 대화 시작**: 하단의 "새 대화 시작" 버튼
- **대화 불러오기**: 기록 클릭
- **대화 삭제**: 호버 시 나타나는 휴지통 아이콘
- **사이드바 숨기기**: 상단 X 버튼

### 검색 결과 (우측 사이드바)

판례 검색 시:
- **판례 탭**: 유사 판례 목록
- **헌재결정례/법령해석례/행정심판례**: 추가 법률 정보
- **판례 상세보기**: 판례 클릭 → 모달 창
- **본문 보기**: "본문" 버튼 클릭

### 피드백 기능

각 AI 응답 하단:
- 👍 좋아요 / 👎 별로예요
- 피드백 코멘트 입력 가능
- 데모에서는 피드백이 저장되지 않습니다

## 기술 데모 포인트

### 프론트엔드

- **Nuxt 3**: 현대적인 Vue.js 프레임워크
- **TypeScript**: 타입 안전성
- **Tailwind CSS**: 반응형 디자인
- **SPA 모드**: 정적 호스팅 최적화

### Mock API

- **Fetch Interception**: 원본 코드 수정 없이 API 가로채기
- **실제 지연 시뮬레이션**: 1.5~2.5초 랜덤 지연
- **현실적인 응답 구조**: 실제 백엔드와 동일한 데이터 형식

### UI/UX

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 대응
- **사이드바 리사이징**: 드래그로 크기 조절
- **마크다운 렌더링**: AI 응답을 읽기 쉽게 포맷
- **로딩 상태**: 실제 API 호출처럼 로딩 애니메이션

## 원본 프로젝트와의 차이

| 기능 | 원본 | 데모 |
|------|------|------|
| AI 분석 | Gemini/GPT 실시간 분석 | 고정 응답 |
| 판례 검색 | OpenSearch 벡터 검색 | Mock 데이터 |
| 파일 업로드 | PDF/DOCX 분석 | UI만 동작 |
| 피드백 저장 | PostgreSQL DB | 동작하지 않음 |
| 로깅 | Langfuse 모니터링 | 콘솔 로그만 |

## FAQ

**Q: 실제로 법률 자문을 받을 수 있나요?**
A: 아니요, 이것은 데모입니다. 실제 법률 자문은 전문 변호사에게 받으세요.

**Q: 왜 모든 질문에 같은 답변이 나오나요?**
A: 데모 모드로, Mock API가 고정된 샘플 응답을 반환하기 때문입니다.

**Q: 원본 프로젝트는 어떻게 다른가요?**
A: 원본은 실제 AI 모델, 벡터 DB, 실시간 분석 기능을 가지고 있습니다.

**Q: 상업적으로 사용 가능한가요?**
A: 이 데모는 포트폴리오 목적입니다. 상업적 사용은 별도 협의가 필요합니다.
